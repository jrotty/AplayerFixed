# AplayerFixed
Aplayer to Typecho背景音乐插件，支持InstantClick与常规pjax的主题

### 功能
- 支持`InstantClick` / `常规Pjax`两种模式
  
- 支持设置播放器位置：左下角或右下角
  
- 支持设置播放顺序：顺序播放或随机
  
- 支持调用网易云音乐歌单：依赖`api.i-meto.com`的`api`
